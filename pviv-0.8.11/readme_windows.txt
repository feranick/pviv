PVIV 0.8.11 - 05/25/2011
----------------------------------------------

1) Description
	
	Pviv converts photovoltaic IV data acquired with GreenEngineering IVStat into ASCII.
	(More into on the IVStat: http://www.greenmountainengineering.com/ivstat).
	Pviv extracts Voc, Jsc, filling factors, and efficiencies from any IV data in ASCII format.

2) Installation
	
	0. Works on Microsoft Windows 2000/XP/Vista/7. DOS/Windows9x are not supported.

	1. Copy the binary provided into any folder that contains the files to be converted.

	2. You can save a copy anywhere on your computer. You can drag and drop files into it for immediate conversion.

	3. You can recompile the binary using Visual Studio Express (the provided bunary was made with VS Express 2010),
	which is freely available from Microsoft. You need to create a blank command line project and import the source pv.cpp file
	into the project. You shoul be able to compile with no issues.

3) Test file

	Several examples can be found in the folder "example".
